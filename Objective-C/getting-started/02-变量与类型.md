## # 条件控制语句

### # if...else...

```

  // ...

  // ...
}
```

### # 高级条件控制语句 goto

缺点：阅读性差

```
int main() {
  @autoreleasepool {
    int i = 0;
    // 做标签
  a:
    i++;
    NSLog(@"i=%d", i);
    if (i < 5) goto a;
  }
  return 0;
}
```

## # 循环语句

### # while

```
int i = 0;
while (i < 5) {
  i++;
  NSLog(@"i=%d", i);
}
```

```
do {
  // ...
} while (表达式)
```

### # for

```
for (int i = 0; i < 10; i++) {
  NSLog(@"i=%d", i);
}
```

### # break语句

跳出循环。

```
for (int i = 0; i < 10; i++) {
  if (i > 4) break;
  NSLog(@"i=%d", i);
}
```


```
int i = 0;
while (i < 5) {
  i++;
  if (i > 2) break;
  NSLog(@"i=%d", i);
}
```

### # continue

```
for (int i = 0; i < 10; i++) {
  if (i % 2 == 0) continue;
  NSLog(@"i=%d", i);
}
```

### # switch case

```
int i = 2;
switch (i) {
  case 1:
    NSLog(@"i = %d", i);
    break;
  default:
    // ...
    NSLog(@"default: %d", i);
    break;
}
```
