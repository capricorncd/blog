# 流程控制语句

### if, if else

```
if (BooleanConditionalExpression) {
  // do something
} else {
  // do something
}

if (BooleanConditionalExpression1) {
  // do something
} else if (BooleanConditionalExpression2) {
  // do something
} else {
  // do something
}
```

### switch

```
switch (Expression) {
  case Value1:
    // do something
    break;
  case Value1:
    // do something
    break;
  default:
    // do something
}
```
