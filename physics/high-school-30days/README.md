### 从物到理的抽象过程

```
观察 -> 抽象 -> 物理量（量化）
```

物理关心的是：通过观察抽象处理的模型，根据该模型能进一步总结处理的"原理"

### 实验

* 可重复

* 控制变量

* 用过于少的数据得出结论是不靠谱的

* 精确度（根据实验水平的不同，得到的结果可能有差异）

|直线1D|2D、3D、4D（竞赛）|
